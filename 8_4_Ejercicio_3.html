El título y este subtítulo tienen que tener background del mismo color

Tenés que usar selectores complejos para lograr que todos los elementos cumplan las siguientes reglas visuales:

  • Los h1 y h2 tengan fondo del mismo color (distinto de blanco).
  • Todos los párrafos tengan un tamaño de letra de 12px.
  • El elemento con id = 'instrucciones' deberá tener un fondo verde, bordes negros de 2 px y tamaño de letra 20px.
  • Los párrafos que sean de clase texto deberán tener un fondo amarillo.
  • Los elementos que sean de clase 'texto' deberán tener un color de texto distinto a los demás.
  • Que los elementos de esta lista tengan bordes rojos:
    1. Los elementos esta sublista tiene que tener fondo rojo.
    2. Tengo fondo rojo!
  • Los links deberán ser de color gris en vez de azul, excepto los que están dentro de un párrafo, esos serán de color cyan
